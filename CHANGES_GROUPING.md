Изменение структуры навигации
=================================

Что сделано:

- Сгруппированы пункты нижней навигации для уменьшения количества вкладок.
  - Новые групповые страницы: `Обучение`, `Соц`, `Админ`, `Вакансии`
- Добавлены компоненты-индексы (простая внутренняя навигация):
  - `src/components/groups/LearningIndex.tsx` — Курсы / Коуч / Резюме
  - `src/components/groups/SocialIndex.tsx` — Закладки / Рейтинг / Геймификация
  - `src/components/groups/AdminIndex.tsx` — Админ-панель и управление
  - `src/components/groups/JobsIndex.tsx` — Вакансии / События
- Навигация обновлена: `src/components/Navigation.tsx` теперь использует группы.
- Подключены групповые страницы в `src/pages/Index.tsx`.

Как запустить локально:

1. Установите зависимости (если нужно):

```bash
npm ci
```

2. Запустите dev‑сервер:

```bash
npm run dev
```

3. Приложение доступно по адресу: http://localhost:8080/

Заметки и дальнейшие шаги:

- В проекте остались ESLint/TypeScript предупреждения и правила, которые требуют внимания (много `any`, некоторые правила хуков). Я временно настроил конфиг так, чтобы можно было запускать dev‑сервер и продолжать итерации.
- Рекомендую в следующем шаге прогнать `npm run lint` и последовательно исправить 10–20 наиболее критичных файлов (замена `any`, исправление условных хук-вызовов и корректные зависимости в `useEffect`). Могу заняться этим по частям.
