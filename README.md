# Проект Navyk

## О проекте

Navyk - платформа для карьерного развития молодых специалистов, включающая систему обучения, менторства, поиска работы и геймификации.

## Разработчик

**Ильяс Азелханов (ElazArt)**

## Технологии

- Vite + React + TypeScript
- Tailwind CSS + shadcn-ui
- Supabase (база данных, аутентификация, edge functions)
- React Query для управления состоянием
- React Router для навигации

## Установка и запуск

1. Клонируйте репозиторий:

```sh
git clone <YOUR_GIT_URL>
cd navyk
npm install
npm run dev
```

2. Настройте переменные окружения в `.env`

## Деплой

Проект можно собрать командой `npm run build` и разместить на любой статической хостинг-платформе (Vercel, Netlify, GitHub Pages и т.д.).
